# Production Control System 
PHPの学習成果物として、生産実績管理システムを作成しました。
  
Herokuでデプロイしました。[PCS](https://pcs51.herokuapp.com/login.php)【登録コード：1234/登録パスワード4321】

## 基本機能
* 文章の投稿  
* 他の人の投稿に返信  
* ユーザ登録  
* 写真をアップロード  
* ログイン  
* ログアウト  
* ページネーション 
* CSV出力
*
## 環境  
* 言語 : PHP 8.0.0
* DB : MariaDB 10.4.17
